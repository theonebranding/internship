# ğŸ” **2.2.8 â€” Recursion (Simple & Powerful)**

A function calling **itself**.

Used for:

* Tree structures (menus, folders, comments)
* Repeating tasks

### âœ” Simple Example (Countdown)

```js
  }
```

### âœ” Real Life Example: Folder Scanner


Intern-level understanding:

> When unknown *how many times* something repeats, recursion is useful.
