# üü¶ **2.1.1 ‚Äî What is Control Flow?**

Control flow = the order in which code executes.

In real life:

* ‚ÄúIf it's raining, take an umbrella.‚Äù
* ‚ÄúElse, wear sunglasses.‚Äù

In JavaScript:

```js
if (isRaining) {
  // take umbrella
} else {
  // sunglasses
}
```

---

# üü© **2.1.2 ‚Äî if Statement**

## ‚úî Basic Structure

```js
if (condition) {
  // code runs when condition is true
}
```

## Example

```js
let age = 20;

if (age >= 18) {
  console.log("You can vote!");
}
```

# üüß **2.1.3 ‚Äî if‚Ä¶else Statement**

```js

  console.log("Pass");
```

Real-life analogy:
‚ÄúIf product is in stock ‚Üí show Buy Now button
Else ‚Üí show Out of Stock.‚Äù

---

# üü• **2.1.4 ‚Äî if‚Ä¶else if‚Ä¶else**

# üü¶ **2.1.5 ‚Äî Comparison Operators**

| Operator | Meaning          | Example           |
| -------- | ---------------- | ----------------- |
| `==`     | equal (loose)    | `5 == "5"` true   |
| `===`    | equal (strict)   | `5 === "5"` false |
| `!=`     | not equal        |                   |
| `!==`    | strict not equal |                   |
| `>`      | greater          |                   |
| `<`      | lesser           |                   |
| `>=`     | greater or equal |                   |
| `<=`     | lesser or equal  |                   |

---

# üü™ **2.1.6 ‚Äî Logical Operators**

Used to combine conditions:

| Operator | Meaning | Example             |    |                   |
| -------- | ------- | ------------------- | -- | ----------------- |
| `&&`     | AND     | both true           |    |                   |
| `        |         | `                   | OR | at least one true |
| `!`      | NOT     | reverses true/false |    |                   |

Example:

```js
let age = 25;
let hasID = true;

if (age >= 18 && hasID) {
  console.log("Entry allowed");
```
```js
}
```

# üü´ **2.1.8 ‚Äî switch Statement**

Used when you have multiple exact options.

Structure:

```js
switch(value) {
  case option1: break;
  case option2: break;
  default: break;
}
```

Example:

```js
let day = 3;

switch(day) {
  case 1: console.log("Monday"); break;
  case 2: console.log("Tuesday"); break;
  case 3: console.log("Wednesday"); break;
  default: console.log("Unknown Day");
}
```

Real-life use:

* Key shortcuts
* Navigation menu
* Choosing theme (light/dark/blue/etc.)

---

# üåü **Real-Life Example: Payment Method Selection**

```js
let payment = "upi";

switch(payment) {
  case "card": console.log("Processing Card Payment..."); break;
  case "upi": console.log("Processing UPI Payment..."); break;
  case "cod": console.log("Cash on Delivery"); break;
  default: console.log("Invalid Payment Method");
```
```js
}
```

# üü† **while Loop**

```js

while (i <= 5) {
  console.log("Hello");
  i++;
}
```

Real-life use: waiting until user logs in.

---

# üî¥ **do‚Ä¶while Loop**

Runs at least once.

```js
let i = 1;

do {
  console.log("This runs once always");
  i++;
} while (i <= 3);
```
