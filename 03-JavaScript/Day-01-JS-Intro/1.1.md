# â— **1.1.3 â€” JavaScript Does NOT Work Outside the Browser (Initially)**

Originally, JS **could only run inside a web browser**.

Meaning:

* You cannot run JS directly in a terminal (without Node).
* You cannot run JS directly on a server.
* You cannot build backend apps with JS.

Chrome, Firefox & other browsers include a **built-in JS engine** that runs your code.

### Without a browser, JS has no environment to run in.

Example:

```js

```

This will NOT run on your PC unless you run it inside:

* A browser console, or
* Node.js

So developers needed a way to run JavaScript **outside browsers**.
