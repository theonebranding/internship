

### üåï **1. What is a Radial Gradient?**

A **radial gradient** creates color transitions radiating from an origin (center or corner) outward in all directions.

Basic syntax:

```css
background: radial-gradient(shape size at position, color1, color2, ...);
```

You can define:

* **shape:** `circle` or `ellipse` (default)
* **size:** `closest-side`, `farthest-side`, `closest-corner`, `farthest-corner`
* **position:** where the gradient starts (like `center`, `top left`, etc.)
* **colors:** at least two color values

---

### üé® **2. Simple Example**

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Radial Gradient Example</title>
  <style>
    .circle {
      width: 300px;
      height: 300px;
      background: radial-gradient(red, yellow);
      border: 2px solid black;
      margin: 30px auto;
    }
  </style>
</head>
<body>
  <div class="circle"></div>
</body>
</html>
```

This creates a circular blend from **red (center)** to **yellow (edges)**.

---

### üí´ **3. Controlling Shape and Size**

By default, the gradient is an **ellipse**, but you can force a perfect **circle**.

```css
background: radial-gradient(circle, red, yellow);
```

You can also define how far the gradient spreads:

```css
background: radial-gradient(circle closest-side, red, yellow);
```

| Size Keyword      | Meaning                                       |
| ----------------- | --------------------------------------------- |
| `closest-side`    | Ends when it hits the closest side of the box |
| `farthest-side`   | Ends when it hits the farthest side           |
| `closest-corner`  | Ends at the closest corner                    |
| `farthest-corner` | (Default) Ends at the farthest corner         |

---

### üß≠ **4. Changing Gradient Position**

You can shift the gradient origin using `at`:

```css
background: radial-gradient(circle at top left, red, yellow, green);
```

Now it starts from the **top-left corner** instead of the center.

---

### üåà **5. Multi-Color Gradients**

Like linear gradients, you can use multiple colors:

```css
background: radial-gradient(circle, red, orange, yellow, green, blue, violet);
```

This produces a rainbow ‚Äúexplosion‚Äù effect from the center outward.

### üß™ **6. Real-Life Examples**

**a. Glowing Button:**

```html
<style>
  .btn {
    padding: 15px 30px;
    border: none;
    border-radius: 8px;
    color: white;
    background: radial-gradient(circle at center, #ff7e5f, #feb47b);
    cursor: pointer;
    transition: 0.3s;
  }
  .btn:hover {
    background: radial-gradient(circle at center, #feb47b, #ff7e5f);
  }
</style>

<button class="btn">Glow Button</button>
```

This gives a smooth, glowing warmth effect often used in modern UI.

---

**b. Spotlight Effect on a Hero Section:**

```html
<style>
  .hero {
    height: 300px;
    background: radial-gradient(circle at center, rgba(255,255,255,0.3), rgba(0,0,0,0.8));
    display: flex;
    align-items: center;
    justify-content: center;
    color: white;
    font-size: 24px;
    font-weight: bold;
  }
</style>

<div class="hero">Spotlight Effect</div>
```

This looks like a soft light shining from the center ‚Äî a simple way to add focus.

---

**c. Abstract Background:**

```html
<style>
  .abstract {
    height: 250px;
    width: 100%;
    background: radial-gradient(circle at 30% 30%, #ff9a9e, transparent 40%),
                radial-gradient(circle at 70% 60%, #fad0c4, transparent 40%),
                radial-gradient(circle at 90% 10%, #fbc2eb, transparent 50%);
  }
</style>

<div class="abstract"></div>
```

By stacking multiple radial gradients, you can make **soft blended shapes** ‚Äî useful for landing pages or section backgrounds.
