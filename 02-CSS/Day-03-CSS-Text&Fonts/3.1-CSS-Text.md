
## üìù What Are CSS Text Properties?

Text properties let you control **how written content appears** on your webpage.
They affect alignment, decoration, spacing, and transformation ‚Äî basically, how readable and expressive your text looks.

---

## üß© Basic Example ‚Äî Text Styling in Action

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>CSS Text Properties</title>
  <style>
    p {
      color: #333;                 /* Changes text color */
      text-align: center;          /* Aligns text: left | right | center | justify */
      text-transform: capitalize;  /* Transforms text case */
      text-decoration: underline;  /* Adds line to text */
      letter-spacing: 3px;         /* Space between letters */
      word-spacing: 6px;           /* Space between words */
      line-height: 1.6;            /* Height of each line */
    }
  </style>
</head>
<body>
  <p>this is a sample paragraph showing css text styling properties.</p>
</body>
</html>
```

## üîç Explanation of Key Text Properties

### 1. **color**

Defines the color of the text.

```css
color: blue;
color: #ff5733;
color: rgb(120, 0, 255);
```

You can use:

* **Color names** (`red`, `blue`, `green`)
* **Hex codes** (`#ff0000`)
* **RGB / RGBA values** (`rgb(255, 0, 0)`, `rgba(255, 0, 0, 0.7)`)

---

### 2. **text-align**

Controls horizontal alignment of text.

```css
text-align: left;     /* Default */
text-align: right;
text-align: center;
text-align: justify;  /* Stretches text to fill full width */
```

Example:

```html
<h2 style="text-align:center;">Centered Heading</h2>
<p style="text-align:justify;">This text will spread evenly across the width of its container.</p>
```

---

### 3. **text-decoration**

Adds decorative lines to text.

```css
text-decoration: underline;
text-decoration: overline;
text-decoration: line-through;
text-decoration: none;
```

You can also combine them:

```css
text-decoration: underline overline;
```

---

### 4. **text-transform**

Changes the capitalization style.

```css
text-transform: uppercase;   /* HELLO WORLD */
text-transform: lowercase;   /* hello world */
text-transform: capitalize;  /* Hello World */
```

---

### 5. **letter-spacing & word-spacing**

Control spacing between letters and words.

```css
letter-spacing: 2px;   /* Adds space between characters */
word-spacing: 8px;     /* Adds space between words */
```

---

### 6. **line-height**

Controls the space between lines of text.
This makes paragraphs easier to read.

```css
line-height: 1.5;     /* Common for paragraphs */
```

---

### 7. **text-indent**

Indents the first line of a paragraph.

```css
text-indent: 50px;
```

---

### üß† Example 2 ‚Äî Combining Text Properties

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Text Example</title>
  <style>
    .quote {
      color: #2c3e50;
      text-align: justify;
      text-transform: capitalize;
      letter-spacing: 2px;
      word-spacing: 5px;
      line-height: 1.8;
      text-indent: 40px;
      text-decoration: none;
    }
  </style>
</head>
<body>
  <p class="quote">
    success is not final, failure is not fatal. it is the courage to continue that counts.
  </p>
</body>
</html>
```

This example shows how all these text properties work together to create beautiful, readable content.

### üß© Example 3 ‚Äî Text Decoration Showcase

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Text Decoration Example</title>
  <style>
    .underline { text-decoration: underline; }
    .overline { text-decoration: overline; }
    .strike { text-decoration: line-through; }
    .combo { text-decoration: underline overline; }
  </style>
</head>
<body>
  <p class="underline">This text is underlined.</p>
  <p class="overline">This text has an overline.</p>
  <p class="strike">This text is struck through.</p>
  <p class="combo">This text is both underlined and overlined.</p>
</body>
</html>
```
