# List Styling

## ðŸ§± Types of Lists

### 1. **Unordered List (`<ul>`)**

Used for bullet points.

```html
<ul>
  <li>Apple</li>
  <li>Banana</li>
  <li>Orange</li>
</ul>
```

### 2. **Ordered List (`<ol>`)**

Used for numbered items.

```html
<ol>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ol>
```

### 3. **Description List (`<dl>`)**

Used for pairs of terms and definitions.

```html
<dl>
  <dt>HTML</dt>
  <dd>Markup language for web pages.</dd>
  <dt>CSS</dt>
  <dd>Styles and layouts for HTML.</dd>
</dl>
```

## ðŸ’¡ Removing Bullets or Numbers

You can remove list markers entirely using `list-style-type: none;`.

```css
ul {
  list-style-type: none;
}
```

You might use this for navigation menus or stylized layouts.

### Example 2: Clean List (No Bullets)

```html
<style>
  ul {
    list-style-type: none;
    padding: 0;
  }

  li {
    background-color: #1e90ff;
    color: white;
    margin: 8px 0;
    padding: 10px;
    border-radius: 6px;
  }
</style>

<ul>
  <li>Home</li>
  <li>About</li>
  <li>Contact</li>
</ul>
```

Now it looks like a menu â€” no default dots, clean spacing.

## ðŸ§± Example 4: Horizontal List (Navigation Style)

Lists are the secret structure behind navbars. We just display items inline.

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Horizontal List</title>
  <style>
    ul {
      list-style-type: none;
      background-color: #333;
      padding: 10px;
      margin: 0;
      text-align: center;
    }

    li {
      display: inline;
      margin: 0 15px;
    }

    a {
      color: white;
      text-decoration: none;
      font-family: "Poppins", sans-serif;
    }

    a:hover {
      color: #1e90ff;
    }
  </style>
</head>
<body>
  <ul>
    <li><a href="#home">Home</a></li>
    <li><a href="#about">About</a></li>
    <li><a href="#contact">Contact</a></li>
  </ul>
</body>
</html>
```

Here:

* `display: inline` puts list items side-by-side.
* `list-style-type: none` removes bullets.
* The list turns into a simple navbar.

## ðŸ§  More About List Properties

| Property              | Description                                | Example                            |
| --------------------- | ------------------------------------------ | ---------------------------------- |
| `list-style-type`     | Changes bullet shape                       | `disc`, `circle`, `square`, `none` |
| `list-style-image`    | Custom bullet image                        | `url("bullet.png")`                |
| `list-style-position` | Places bullet inside or outside text block | `inside`, `outside`                |
| `list-style`          | Shorthand for all above                    | `list-style: square inside;`       |
