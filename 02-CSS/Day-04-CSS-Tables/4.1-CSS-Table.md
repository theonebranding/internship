# Table with styling

## üß± Basic HTML Table

Before styling, here‚Äôs a simple table to work with:

```html

  <tr>
    <th>Name</th>
    <th>Position</th>
    <th>Salary</th>
  </tr>
  <tr>
    <td>Rahil</td>

  </tr>
  <tr>
    <td>Aisha</td>
    <td>Designer</td>
    <td>$1800</td>
  </tr>
</table>
```

By default, HTML tables have no style ‚Äî they look flat and boring.
Let‚Äôs use CSS to fix that.

---

## üé® Basic Table Styling

```css
table {
  width: 100%;
  border-collapse: collapse; /* merges borders together */
}

th, td {
  border: 1px solid black;
  padding: 8px; /* space inside cell */
  text-align: left;
}

th {
  background-color: #f2f2f2;
}
```

### Output Description:

* `border-collapse: collapse;` makes the table look cleaner (no double borders).
* `padding` adds breathing room inside cells.
* `background-color` on `<th>` makes headers distinct.

---

## üìê Important CSS Table Properties

| Property           | Description                                                     | Example                        |
| ------------------ | --------------------------------------------------------------- | ------------------------------ |
| `border`           | Adds a line around cells or the table                           | `border: 1px solid #333;`      |
| `border-collapse`  | Merges adjacent borders                                         | `border-collapse: collapse;`   |
| `border-spacing`   | Sets space between cells (only works if collapse is *not* used) | `border-spacing: 10px;`        |
| `width` / `height` | Controls table size                                             | `width: 80%; height: 200px;`   |
| `padding`          | Space inside each cell                                          | `padding: 10px;`               |
| `text-align`       | Aligns text horizontally                                        | `text-align: center;`          |
| `vertical-align`   | Aligns text vertically                                          | `vertical-align: middle;`      |
| `background-color` | Cell background                                                 | `background-color: lightblue;` |
| `color`            | Text color                                                      | `color: #333;`                 |

## üß© Example: Alternating Row Colors (Zebra Table)

```css
tr:nth-child(even) {
  background-color: #f9f9f9;
}

tr:nth-child(odd) {
  background-color: #ffffff;
}

tr:hover {
  background-color: #e0e0e0;
}
```

This gives your table a **striped effect** and a hover highlight.
It‚Äôs a simple trick used in dashboards and admin panels.

## üß† Example: Styled Table with Borders and Rounded Corners

```html
<table class="styled-table">
  <tr>
    <th>Student</th>
    <th>Subject</th>
    <th>Marks</th>
  </tr>
  <tr>
    <td>Ravi</td>
    <td>Math</td>
    <td>89</td>
  </tr>
  <tr>
    <td>Sara</td>
    <td>Science</td>
    <td>92</td>
  </tr>
</table>
```

```css
.styled-table {
  width: 80%;
  border-collapse: collapse;
  margin: 20px 0;
  border-radius: 8px;
  overflow: hidden;
  box-shadow: 0 0 10px rgba(0,0,0,0.1);
}

.styled-table th {
  background-color: #4CAF50;
  color: white;
  padding: 12px;
}

.styled-table td {
  border: 1px solid #ddd;
  padding: 10px;
}

.styled-table tr:hover {
  background-color: #f1f1f1;
}
```

### What‚Äôs New Here:

* `border-radius` ‚Üí rounds the table edges.
* `box-shadow` ‚Üí gives a soft shadow (3D effect).
* `overflow: hidden` ‚Üí hides content spilling past the rounded edges.

## üîç Comparison: `border-collapse` vs `border-spacing`

**Without collapse:**

```css
table {
  border-collapse: separate;
  border-spacing: 10px;
}
```

You‚Äôll see gaps between cells.

**With collapse:**

```css
table {
  border-collapse: collapse;
}
```

No gaps ‚Äî cells share the same borders.

---

## üß≠ Summary

* Use **`border-collapse: collapse`** for a clean, tight layout.
* Add **`padding`** to improve readability.
* Use **row striping** and **hover effects** for better UX.
* Combine **borders**, **radius**, and **shadows** for a modern look.
