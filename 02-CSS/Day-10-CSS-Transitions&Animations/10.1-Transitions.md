# âš™ï¸ **Parts of a Transition**

| Part              | Meaning                  | Example                                    |
| ----------------- | ------------------------ | ------------------------------------------ |
| `property`        | What to animate          | `background-color`, `transform`, `opacity` |
| `duration`        | How long (in seconds/ms) | `0.5s` or `500ms`                          |
| `timing-function` | How the motion feels     | `ease`, `linear`, `ease-in`, `ease-out`    |
| `delay`           | When to start            | `0.2s` (optional)                          |

You can transition multiple properties together:

```css
transition: background-color 0.4s ease, transform 0.4s ease;
```

# ğŸ’¡ **Timing Functions**

They decide *how fast* your animation moves at the start and end.

| Function            | Description                              | Example Feel         |
| ------------------- | ---------------------------------------- | -------------------- |
| `linear`            | Same speed all along                     | loading bar          |
| `ease`              | Starts slow, speeds up, then slows again | natural button hover |
| `ease-in`           | Starts slow, ends fast                   | entering animation   |
| `ease-out`          | Starts fast, ends slow                   | exiting/fade-out     |
| `ease-in-out`       | Slow start and end                       | elegant              |
| `cubic-bezier(...)` | Custom curve                             | advanced control     |
