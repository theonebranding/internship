# âš™ï¸ **Animation Shorthand (Easy Formula)**

You can set all animation options in one line:


Example:

```css

```

| Term      | Meaning           | Example            |
| --------- | ----------------- | ------------------ |
| name      | which keyframes   | `moveRight`        |
| duration  | how long          | `3s`               |
| timing    | how speed changes | `ease-in-out`      |
| delay     | wait before start | `1s`               |
| iteration | how many loops    | `infinite`, `3`    |
| direction | forward/backward  | `alternate`        |
| fill-mode | keep end state?   | `forwards`, `both` |
