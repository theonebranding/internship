
## üéØ What Is `z-index`?

`z-index` controls the **stacking order** of elements on the **z-axis** (the imaginary axis that comes out of your screen).

Think of it as **who sits on top of whom** when elements overlap.

```css
z-index: 1;
```

The higher the number, the closer it appears to the viewer.

---

## ‚öôÔ∏è Basic Rule

`z-index` **only works** on elements that have a **position** other than `static`.

So, you must use:

```css
position: relative;
position: absolute;
position: fixed;
position: sticky;
```

before `z-index` can take effect.

---

## üß© Example 1: Basic Layering

```html
<div style="position: relative; width: 300px; height: 200px; background: lightblue;">
  <div style="position: absolute; top: 40px; left: 40px; width: 100px; height: 100px; background: tomato; z-index: 1;"></div>
  <div style="position: absolute; top: 70px; left: 70px; width: 100px; height: 100px; background: gold; z-index: 2;"></div>
  <div style="position: absolute; top: 100px; left: 100px; width: 100px; height: 100px; background: lime; z-index: 3;"></div>
</div>
```

**Explanation:**

* All three small boxes overlap.
* The one with the **highest `z-index` (3)** appears **on top**.
* Think of them as colored cards stacked in layers.

---

## üß± Example 2: Without `z-index`

```html
<div style="position: relative; width: 250px; height: 150px; background: skyblue;">
  <div style="position: absolute; top: 30px; left: 30px; width: 100px; height: 100px; background: salmon;"></div>
  <div style="position: absolute; top: 60px; left: 60px; width: 100px; height: 100px; background: orange;"></div>
</div>
```

Here, both boxes overlap, but since no `z-index` is given, the **later element in HTML** (orange) automatically sits on top.
That‚Äôs the **default stacking order**.

## üß† Real-Life Use Cases

1. **Modals or Popups**

   ```css
   .modal {
     position: fixed;
     top: 0;
     left: 0;
     width: 100%;
     height: 100%;
     background: rgba(0, 0, 0, 0.5);
     z-index: 1000;
   }
   ```

   The modal appears **above** all page content.

2. **Dropdown Menus**

   ```css
   .dropdown {
     position: absolute;
     background: white;
     z-index: 10;
   }
   ```

   Ensures the dropdown menu doesn‚Äôt hide behind other elements.

3. **Sticky Navigation Bars**

   ```css
   .navbar {
     position: sticky;
     top: 0;
     background: #333;
     color: white;
     z-index: 999;
   }
   ```

   Keeps your navbar floating over scrolling content.

## üß© Example 3: z-index Chaos (and Why Order Matters)

```html
<div style="position: relative; width: 300px; height: 200px; background: #eceff1;">
  <div style="position: absolute; top: 20px; left: 20px; width: 100px; height: 100px; background: red; z-index: 5;"></div>
  <div style="position: absolute; top: 50px; left: 50px; width: 100px; height: 100px; background: blue; z-index: 2;"></div>
  <div style="position: absolute; top: 80px; left: 80px; width: 100px; height: 100px; background: green; z-index: 8;"></div>
</div>
```

* Red: `z-index: 5`
* Blue: `z-index: 2`
* Green: `z-index: 8` (on top of all)

If you remove their `z-index`, their order in HTML will control the layering again.

## üß¨ Stacking Context (Slightly Advanced but Crucial)

Each positioned element **creates its own stacking context** ‚Äî like a *mini world* of z-index values.

Example:

```html
<div style="position: relative; z-index: 1;">
  <div style="position: relative; z-index: 999;">Child</div>
</div>
<div style="position: relative; z-index: 10;">Outside</div>
```

Even though the **child** has `z-index: 999`, it still stays **behind** the second box (`z-index: 10`) because its parent (`z-index: 1`) creates a new stacking context.

So `z-index` values are only compared **within the same context**.

---

## üìê Example 4: Combine Position + Overflow + z-index

```html
<div style="position: relative; width: 300px; height: 150px; background: #c8e6c9; overflow: hidden;">
  <img src="https://via.placeholder.com/300x150" style="position: absolute; top: 0; left: 0; z-index: 1;">
  <div style="position: absolute; top: 0; left: 0; width: 100%; height: 100%; background: rgba(0,0,0,0.4); z-index: 2;"></div>
  <h3 style="position: absolute; bottom: 10px; left: 10px; color: white; z-index: 3;">Beautiful Overlay</h3>
</div>
```

This is how you create **overlay effects** ‚Äî images, dark transparent layers, and text stacked in the right order.
